# NeuroForge

A C++ neural network library for learning advanced C++ features while building scalable AI tools.

## Getting Started
- Build: `mkdir build && cd build && cmake .. && make`
- Run: `./neuroforge`

## Folder Structure
- src/: Source code
- include/: Headers
- tests/: Unit tests
